For the first tree I did a maximum likelihood method. I did 1000 bootstraps coming up with a 95% accuracy. This type of tree uses all the data that is given. It results in a tree that is maximized to have the specific parameter values. It is also great for small data sets. For the second tree I did a neighbor-joining method. I did 10000 bootstraps coming up with a 79% accuracy. This method uses the pairwise distances. This is better when used with larger data sets. Bootstrapping is basically a metric used to use random sampling in order to find out the best results. It makes a bunch of different trees to see how many times it gets the same answer, telling us which is the most accurate.
